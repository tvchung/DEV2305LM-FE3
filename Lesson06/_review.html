Review - CSS-1

            1. Nhúng css vào html 
                - style inline 
                - tag style (trên phần head của trang html)
                - External file style (file.css -> link / import)
            2. CSS selector 
                * Bộ chọn phần tử
                    tagHTML {....} 
                    ex: h1{...} p{....}
                * Bộ chọn ID:
                    #id {....} 
                    ex: #zone1{background-color:#ccff12;}
                * Bộ chọn class
                    .class-name {....}
                    ex: 
                        .noi-dung{
                            color:blue
                        }
                * Bộ chọn theo giá trị của thuộc tính 
                    tag[attribute=value]{.....} 
                    ex:
                        input[type="text"]{
                            border:1px solid #ccc;
                        }
                * Bộ chọn ngữ cảnh
                    tag:hover 
                    tag:active 
                    tag:before / tag:after 
                    tag:first-child ; ... 
                    .... 
                    ex: 
                        a:hover{
                            color:red
                        }
                * Bộ chọn: 
                    - tag01, tag02 {... }
                    - tag01 > tag02 {....} 
                    - tag01 ~ tag02 {.... }
                    - tag01 + tag02 {.... }
                    ..... 
                * ....
